# BrunoWebTeste

**Ferramenta de Teste de Estresse para Fins Didáticos**

**O BrunoWebTeste é uma ferramenta simples de teste de estresse desenvolvida com o propósito exclusivo de aprendizado e demonstração didática. Ela foi criada para ilustrar conceitos básicos de automação de testes e testes de estresse em ambientes web.**

# Funcionalidades

1. **Teste Automatizado:** Automatize a interação com uma página web específica, simulando recarregamentos e avaliando condições.

2. **Número de Iterações Configurável:** Escolha o número desejado de iterações para o teste.

3. **Delay Personalizável:** Configure o intervalo de tempo entre as iterações.

4. **Ativação/Desativação de Scroll Automático:** Controle se a página realiza automaticamente um scroll até o final após cada recarregamento.

5. **Verificação de Condição:** Adapte o teste para verificar uma condição específica durante a execução.

6. **Acompanhamento do Progresso:** Visualize o progresso do teste e o tempo decorrido.
